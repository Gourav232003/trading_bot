# Binance Futures Testnet Trading Bot

A simple Python CLI trading bot for Binance USDT-M Futures Testnet.

## Features
- Market & Limit orders
- BUY / SELL support
- CLI-based input
- Logging to file
- Input validation & error handling
- Clean modular structure
